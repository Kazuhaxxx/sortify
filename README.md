# Sortify – AI-Powered Waste Classification

## Overview
**Sortify** is an AI-powered waste classification system designed to improve public participation in waste separation at source in Malaysia.

The system uses Deep Learning and Transfer Learning (MobileNetV2) to classify solid waste images into the following categories:
* Cardboard
* Glass
* Metal
* Paper
* Plastic

This project supports Malaysia’s National Solid Waste Management Policy (DPSPN) and aligns with UN Sustainable Development Goal 12 (Responsible Consumption and Production).
